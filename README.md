# 店舗管理システム

## 機能
 ### 1.ログイン機能
 
 ・従業員新規登録は管理者のみ可能
 
 ### 2.勤怠管理機能
 ◉タイムカード機能
 ・ログイン従業員の出勤、退勤、休憩開始、休憩終了時間を打刻可能
 
 ・同じ日に2度出勤打刻が出来ない仕様
 
 ・退勤打刻後、就労時間を計算し表示
 
 ◉勤怠実績
 
 　・ログイン従業員の勤務実績を選択した月ごとに表示
  
 ◉日次勤怠
 
 　・選択した日次に打刻された全従業員の勤怠実績を表示

 ### 3.連絡ノート
 
   ・作成、削除、編集機能

 ### 4.発注画面
　　◉発注入力
  
   ・1日1回のみ発注入力可能（重複発注禁止)
   
　 ・在庫数を入力することにより、発注数を計算しデータベースに発注数を登録
  
   ◉発注履歴
   
   ◉発注アイテム追加
